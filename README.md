# propositional-formula-processor
Propositional Formula Processor is a desktop application developed as a university project for processing and analyzing propositional logic expressions.

The application allows users to enter logical formulas, transform them into different representations, evaluate them under various interpretations, and visualize their structure.

| Symbol | Meaning    |
| ------ | ---------- |
| ¬      | NOT        |
| ∧      | AND        |
| ∨      | OR         |
| →      | IMPLIES    |
| ↔      | EQUIVALENT |


Features
- Formula input through GUI
- Infix to postfix conversion
- Formula simplification
- Conjunctive Normal Form (CNF/FNC) transformation
- Truth-value evaluation
- Parse tree generation
